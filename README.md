# Stash Buster

A web application for knitters and crocheters to manage yarn stashes and pattern collections.

## Tech Stack
- React (frontend)
- Node.js + Express (backend)
- MongoDB (database)

## Core Features
- Yarn stash inventory
- Pattern library
- Yarn-to-pattern matching
- User authentication
